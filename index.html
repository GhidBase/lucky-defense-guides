Beating Normal and Hard Mode

Beating normal mode is one of the first barriers in the game to overcome. While these guides I've listed below are written specifically with hard mode in mind, if you read these you will not only win your games, but you'll progress in a way that prepares you for hard mode as well. If you apply hard mode strategies to normal, very quickly normal mode will become easy mode.

The most important parts

I reccomend reading at least these four guides about game mechanics if you want to start winning every normal mode match you play.

  • How to use stuns
  • Mythic Unlock Order
  • Defense Reduction
  • The most powerful artifacts in the game

Reading mythic roles will also help.
